Política de Privacidade — Bolão Copa 2026

Última atualização: 25 de maio de 2026

O Bolão Copa 2026 é um aplicativo de bolão esportivo para a Copa do Mundo 2026, desenvolvido para uso entre grupos de amigos. Esta política descreve quais dados são coletados, como são utilizados e com quem são compartilhados.

1. Dados coletados

  • E-mail e nome de usuário — fornecidos no cadastro, usados para identificação e exibição no ranking.
  • Palpites de placar, campeão e artilheiro — inseridos pelo próprio usuário e exibidos para os demais participantes do grupo.
  • Avatar selecionado — foto de jogador escolhida pelo usuário para personalização do perfil.
  • Pontuação e posição no ranking — calculadas automaticamente com base nos palpites e resultados reais.
  • Token FCM — identificador do dispositivo, usado exclusivamente para o envio de notificações push (lembretes de palpite e mudanças no ranking).

2. Dados NÃO coletados

O aplicativo não coleta localização, contatos, câmera, microfone, dados bancários, documentos pessoais nem qualquer outro dado sensível.

3. Uso dos dados

Os dados coletados são usados exclusivamente para:

  • Exibir o ranking e os palpites entre os participantes do mesmo grupo;
  • Enviar notificações push de lembrete antes de jogos e de atualização de ranking.

Os dados não são utilizados para fins comerciais, publicidade ou venda a terceiros.

4. Compartilhamento de dados

Os dados são armazenados no Google Firebase (Firestore, Authentication e Cloud Messaging), serviço gerenciado pelo Google LLC. Nenhum dado é compartilhado com outras empresas ou pessoas além dos participantes do mesmo grupo dentro do aplicativo. A política de privacidade do Google está disponível em policies.google.com/privacy.

5. Retenção e exclusão de dados

Os dados são mantidos enquanto o usuário possuir conta ativa no aplicativo. O usuário pode excluir sua conta a qualquer momento pela tela de perfil do app, o que remove permanentemente todos os seus dados do Firestore e da autenticação Firebase.

6. Segurança

O acesso ao banco de dados é protegido por regras de segurança do Firestore, garantindo que cada usuário acesse apenas os dados pertinentes ao seu grupo. A comunicação entre o app e o servidor utiliza HTTPS.

7. Crianças

O aplicativo não é direcionado a crianças menores de 13 anos e não coleta intencionalmente dados de menores.
